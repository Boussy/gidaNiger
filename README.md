# gidaNiger

1. Vision
Créer une plateforme SaaS mobile et web dédiée à la diaspora nigérienne (Niger). Elle
centralise l'entraide, les opportunités, les informations fiables et les échanges
communautaires.

2. Problème
Les échanges sont aujourd'hui dispersés entre WhatsApp et Facebook. Les offres
d'emploi, logements, conseils administratifs et événements sont difficiles à retrouver.

3. Proposition de valeur
Une plateforme géolocalisée avec un fil d'actualité type Twitter permettant de publier
des posts classés par catégories et par ville.

4. Fonctionnalités
Libellés : Emploi, Logement, Business, Solidarité, Administration, Juridique, À donner.
Extensions : Études, Événements, Transport, Recommandations, Urgence.

5. UX/UI
Interface moderne inspirée de Twitter/X avec navigation simple : Accueil, Explorer,
Publier, Notifications, Profil. Palette : Orange #CB6D30, Vert #2B4533, Beige
#F0E6D9. Mode sombre : fond #1E1E1E.

6. Étude de marché
Diaspora plus réduite que celle de pays voisins mais organisée. Besoins identifiés :
emploi, logement, réseau, démarches administratives et solidarité. Sources : OIM,
UNDESA, Banque mondiale, rapports Diaspora for Development, Nigerdiaspora.net,
ambassades.

7. Business model
Freemium pour les particuliers. Revenus via comptes premium, visibilité des annonces,
publicité ciblée, sponsoring et comptes professionnels.

8. Architecture technique
Frontend : React Native (Expo) + Next.js. Backend : NestJS. API REST (GraphQL
possible). PostgreSQL + Prisma. Supabase ou AWS. Stockage S3/Supabase. Auth
JWT/OAuth. Firebase Cloud Messaging. GitHub Actions, Expo EAS, Vercel. Analytics
PostHog/Firebase. Monitoring Sentry.

9. Roadmap
Validation du besoin -> Maquettes Figma -> Développement MVP -> Bêta sur quelques
villes -> Lancement officiel -> Évolutions.

10. Facteurs clés
Créer rapidement une communauté, commencer par quelques villes pilotes, offrir une
excellente expérience utilisateur, développer progressivement la monétisation.

11. Conclusion
GidaNiger vise à devenir la plateforme de référence de la diaspora nigérienne en
centralisant les échanges et les services dans une application moderne.
